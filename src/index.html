How to Use Textlet
  • Bold: Select text and click the button.
  • Italic: Select text and click the button.
  • Underline: Select text and click the button.
  • Strikethrough: Select text and click the button.
  • Align Left: Click the button.
  • Center: Click the button.
  • Align Right: Click the button.
  • Font Size: Select a size from the dropdown menu.
  • Text Color: Pick a color from the color input.
  • Add Link: Enter a URL and select text, then click the button.
